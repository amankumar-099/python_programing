# Python Beginner Projects – Utility Scripts

This repository contains a collection of beginner‑friendly Python programs that demonstrate conditional logic, input handling, and practical applications of programming. Each script solves a simple real‑world problem, making them excellent practice exercises for learners.

## Included Programs

### 1. Password Strength Checker
Validates password strength based on length, uppercase and lowercase letters, digits, and special characters. Provides feedback on missing criteria to help users create secure passwords.

### 2. Age Approval for Driving
Determines driving eligibility based on age input. Approves users aged 18 and above, rejects those younger, and includes checks for unrealistic values like negative ages or excessively high numbers.

### 3. Loan Approval System
Evaluates loan eligibility based on monthly salary. Categorizes applicants into high, medium, or small loan brackets and calculates the maximum loan amount (e.g., 5×, 8×, or 10× salary). Provides clear approval status messages.

### 4. Student Performance Card
Generates a grade and remark based on overall percentage input. Grades range from A+ (Excellent) to F (Fail), with personalized feedback to encourage improvement.
